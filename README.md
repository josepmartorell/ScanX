Installation
------------

Install ScanX is so easy:

```
$ python -m pip install ScanX
```

**Remember that ScanX only runs in Python 3**.

Quick start
-----------

You can display inline help writing:

```bash

python ScanX.py -h
```

Advanced options
----------------

There are the advanced options:

- **-v**, **-vv**, **-vvv**: Enable verbose mode.
